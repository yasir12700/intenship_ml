# Titanic Dataset - Exploratory Data Analysis 
This project provides an in-depth exploratory data analysis  of the Titanic passenger dataset, which is a classic beginner-friendly dataset in the field of data science and machine learning. 

## Objective

To understand the structure and key relationships within the Titanic dataset using statistics and data visualizations. The insights gathered from this exploratory data analysis will help in building predictive models .


##  Tools & Libraries Used
- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`
1. **Summary Statistics**
   - Data types
   - Missing values
   - Mean, median, standard deviation

2. **Visual Analysis**
   - Histograms for distributions
   - Boxplots for outlier detection
   - Heatmap for correlation matrix
   - Pairplots for feature interaction
   - Categorical feature analysis (e.g., Gender, Pclass)

3. **Insights & Inferences**
   - Survival patterns by gender and class
   - Age and fare trends
   - Feature-level observations useful for ML modeling
